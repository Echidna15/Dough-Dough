# Dough Dough - Handmade Bakery Website

A beautiful, modern website for Dough Dough, a bakery specializing in handmade sweet and savory baked goods.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Product Gallery**: Interactive image gallery showcasing all handmade products
- **Sweet & Savory Sections**: Organized product categories for easy browsing
- **Contact Form**: Easy way for customers to get in touch
- **Smooth Animations**: Beautiful scroll animations and transitions
- **Image Gallery Modal**: Click on any product to view multiple images

## Product Images

The website displays products with images numbered 1-9:
- Products 1-6: Sweet treats (cakes, cookies, pastries, muffins, donuts, brownies)
- Products 7-9: Savory delights (breads, pies, quiches)

Each product can have multiple images (main, second, third) that can be viewed in the gallery modal.

## Technologies Used

- HTML5
- CSS3 (with CSS Variables for theming)
- Vanilla JavaScript
- Google Fonts (Playfair Display & Poppins)

## Getting Started

1. Open `index.html` in a web browser
2. Navigate through the sections using the navigation menu
3. Click on any product card to view the image gallery
4. Use arrow keys or navigation buttons to browse through product images

## Customization

- Update contact information in the Contact section
- Modify product descriptions in the HTML
- Adjust colors in the CSS `:root` variables
- Add or replace product images (follow the naming convention: `[number] [type].jpg`)
